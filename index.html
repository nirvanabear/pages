Howdy, all!

I've been living in the east bay for the past 15 years. I currently work at a plumbing supply store in Marin county and I've been taking coding classes at CCSF off and on for about 5 years. I've stuck mostly with programming, learning things like SQL, Python, C++, HTML/CSS and Linux shell script. Eventually I'd like to be working in the software development field. Not exactly sure where, but that is going to be a big project of mine in the next couple of months, figuring out which way I want to go. Web backend, web page design, application development, even a little machine learning. I've enjoyed bits of all of it.
