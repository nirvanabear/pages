Hi!

My name is George, and I work at a plumbing supply store in Marin County. I live in the east bay and have lived here for the past 13 years after moving here to finish my degree at UC Berkeley. I have been learning computer programming for the past five years through online CCSF classes while still working full time.

I am currently working on building out pages for my very first website. I've done some backend work before on a project called We Vote, which is a non-profit website designed to help voters make choices aligned with their values. I worked on creating a unit test in python for one of its functions. The project involved forking the existing GitHub repository, cloning it to an AWS server, and setting up a virtual environment and environment variables in order to get a development version of the site up and running. This will be my first time creating front-end content.
